# Knowledge-hub

## Description

Knowledge-hub spa application build with:
- javascript
- python
- html, css, bootstrap css

### Features
#### Dashboard app follow the information inside, consist of:

- DEOC Trend
- DEOC Top Defect
- LINE Trend
- SKU Trend
- Insight

#### Safety chat

- Safety info
- Safety advise

### Requirements:
        pip install -r requirements.txt

### Test:
        python main.py
